# python-honeypot-poc
HoneyPot de baixa interatividade desenvolvido em Python para monitoramento de redes locais. Captura tentativas de conexão HTTP, registra metadados de intrusão em arquivo de log e emite alertas sonoros e visuais (PoC). Ferramenta didática para estudos de defesa cibernética e detecção de ameaças.
